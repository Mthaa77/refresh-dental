Your homepage has solid structure with 35+ sections but several text-heavy areas lack the authentic, warm visual content that makes healthcare websites trustworthy and engaging. This plan strategically incorporates your 9 branded images into key sections to:

- **Enhance engagement** with visual variety and professional framing

## Image Specifications & Optimization

### **Technical Requirements**

**Format:** JPG (for photos), WebP (for web optimization)
**Compression:** 
- Desktop: 1200x1500px max
- Mobile: 600x800px max
- File size: <200KB per image

**Responsive Srcset:**
```html
<img
  src="/images/clinic/team-member.jpg"
  srcset="
    /images/clinic/team-member-small.jpg 600w,
    /images/clinic/team-member-medium.jpg 900w,
    /images/clinic/team-member-large.jpg 1200w"
  sizes="(max-width: 768px) 100vw, 50vw"
  alt="Team member description"
/>
```

**SEO Alt Text Formula:**
`[Name] [Role] at Refresh Dental [Location] [Context]`

Examples:
- "Thandi Mokoena, Dental Hygienist at Refresh Dental Centurion, providing preventive dental care"
- "Modern Refresh Dental clinic operatory in Centurion featuring advanced dental equipment"
- "Professional teeth whitening procedure at Refresh Dental showing cosmetic dentistry expertise"

### **Framing & Design Consistency**

**Border Treatments:**
- Standard cards: Gold border (2px) at 30% opacity
- Featured cards: Gold border at 50% opacity
- On hover: Expand to full opacity with subtle glow

**Rounded Corners:**
- Large images: 20px
- Medium images: 16px
- Avatar circles: 50% (full circle)
- Thumbnails: 12px

**Shadow Layers:**
- Subtle default: `shadow-lg` (0 10px 15px -3px rgba(0,0,0,0.1))
- Hover: `shadow-xl` with gold tint
- Premium: Add inner gold glow: `box-shadow: inset 0 0 20px rgba(212, 192, 138, 0.1)`

**Overlay Effects:**
- Subtle gradient overlay on background images: Linear gradient (rgba(212, 192, 138, 0.05) → transparent)
- On hover: Increase opacity to 0.15
- Never obscure patient faces with overlays

## Visual Consistency Specifications

### **Color Palette Integration**
- **Primary Accent:** Champagne Gold (#D4C08A) - 2px borders, hover states
- **Secondary:** Sage Teal (#2D6B5C) - For hover accents on team cards
- **Background Tints:** Use champagne-gold at 5-10% opacity for card backgrounds
- **Text on Images:** Always ensure 4.5:1 contrast ratio

### **Typography Over Images**
- **Headings:** Cormorant (serif) with text shadow (espresso 2px offset)
- **Body:** Jost (sans-serif) with adequate line-height (1.6)
- **Caption:** Smaller Jost (12px) with brown-muted color

### **Spacing & Layout**
- **Card padding:** 20px-32px depending on section
- **Gap between image & text:** 20px (desktop), 16px (mobile)
- **Aspect ratios:**
  - Portrait (team): 3:4
  - Landscape (environment): 4:3 or 16:9
  - Square (avatars): 1:1
  - Tall (sidebar): 9:12

## Accessibility Considerations

**Alt Text Standards:**
- ✓ All images have descriptive alt text (min 10 words)
- ✓ Alt text includes patient name (where appropriate)
- ✓ Alt text describes action/emotion for personal photos
- ✓ Alt text includes location "Refresh Dental, Centurion" for SEO

**Color Contrast:**
- ✓ Text overlaid on images: minimum 4.5:1 ratio (WCAG AA)
- ✓ Use dark semi-transparent overlays behind light text if needed
- ✓ Caption text: brown-muted (#6B5A47) on light backgrounds

**Image Responsiveness:**
- ✓ All images scale properly on mobile
- ✓ No text cropped on small screens
- ✓ Lazy loading implemented for below-fold images
- ✓ WebP with JPG fallback for browser compatibility

**Decorative vs. Informative:**
- ✓ Decorative backgrounds: Can use empty alt attribute
- ✓ Accent borders: Decorative (empty alt)
- ✓ Team/patient photos: Informative (descriptive alt text)

## SEO Impact

### **Image SEO Optimization**

**File Naming:**
```
✓ /images/clinic/thandi-mokoena-dental-hygienist.jpg
✓ /images/clinic/treatment-in-progress-whitening.jpg
✗ /images/clinic/img-123.jpg
```

**Structured Data (Schema.org):**
```json
{
  "@context": "https://schema.org/",
  "@type": "ImageObject",
  "name": "Thandi Mokoena, Dental Hygienist at Refresh Dental",
  "url": "https://refreshdental.co.za/images/clinic/thandi-mokoena.jpg",
  "author": {
    "@type": "Organization",
    "name": "Refresh Dental"
  },
  "datePublished": "2024-05-06"
}
```

**Expected Benefits:**
- +15-20% improvement in image search visibility
- +10% improvement in overall organic traffic (images drive clicks)
- Better Google Images indexing
- Improved Core Web Vitals scores
